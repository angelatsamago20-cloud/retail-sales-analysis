# Retail Sales Analysis Project

## Objective
Analyze a Kaggle retail sales dataset using SQL to uncover trends, top-performing products, and customer insights.

## Tools Used
- SQL(for data analysis)
- GitHub (for project hosting and portfolio)

## Key Questions
- How many records are in the dataset?
- What is the date range of the data?
- Which product categories generate the highest total sales?
- Which product categories are the most profitable?

## Key Findings
- Total records in the dataset: 9994
- Date range: 2014-01-03 to 2017-12-30
- Top-selling category: generated the highest total sales, making it the strongest-performing category by revenue, is Technology (Total sales: 836154.03)
- Most profitable category: Technology with a total profit of  145454.95

- ## Dataset
- Source: Kaggle (public dataset)
- Format: CSV
- Description: Retail sales transaction data used to analyze sales and profitability across product categories.
